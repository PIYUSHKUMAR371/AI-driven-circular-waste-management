# ♻️ ClearSightAI

### AI-Driven Circular Waste Management System

🚀 A smart, scalable platform that uses **AI + Geospatial Intelligence** to optimize waste collection, reduce costs, and promote sustainable urban living.

---

## 🌍 Overview

ClearSightAI is designed to tackle inefficient waste collection systems by introducing:

* 📍 Smart bin tracking
* 🧠 AI-based route optimization
* 🚛 Real-time fleet monitoring
* 📊 Data-driven decision making

The platform ensures **minimum travel distance**, **reduced fuel usage**, and **efficient waste handling**.

---

## ✨ Key Features

### 🗺️ Intelligent Route Optimization

* Dynamically calculates the **most efficient path** for waste collection
* Adapts automatically when new bins are added

### 🧠 AI Waste Monitoring

* Tracks bin fill levels
* Detects anomalies in waste patterns

### 📡 Live Fleet Tracking

* Monitor garbage trucks in real-time
* Improve operational transparency

### 📊 Admin Dashboard

* View analytics, routes, and performance metrics
* Manage bins and logistics efficiently

### 🌦️ Weather Integration

* Handles disruptions like bad weather
* Adjusts routes accordingly

## 🚀 Getting Started

### 🔹 Clone the repository

```
git clone https://github.com/PIYUSHKUMAR371/AI-driven-circular-waste-management.git
cd ClearSightAI
```

---

### 🔹 Install dependencies

```
npm install
```

---

### 🔹 Run frontend

```
npm start
```

---

### 🔹 Run backend

```
uvicorn main:app --reload
```

---

## 🔐 Environment Variables

Create a `.env` file:

```
WEATHER_API_KEY=your_api_key_here
MONGO_URI=your_mongodb_connection
```

⚠️ Do NOT expose this file publicly.
